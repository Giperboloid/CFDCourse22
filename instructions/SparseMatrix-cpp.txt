Для решения СЛАУ в С++ использовать amgcl.

1) Cкачать архив amgcl
https://github.com/ddemidov/amgcl/archive/refs/tags/1.4.2.zip

2) Скачать boost. Последняя на сейчас версия здесь
https://boostorg.jfrog.io/artifactory/main/release/1.78.0/source/boost_1_78_0.zip

3) Дальше нужно распаковать эти архивы в какую-то папку. Например C:/lib/
Чтобы в этой папке были директории c содержанием архивов
amgcl-1.4.2\*      {внутри неё должна быть в том числе папка amgcl\} 
boost_1_78_0\*     {внутри неё должна быть в том числе папка boost\}

4)
При компилляции кода нужно добавить эти две папки в каталоги включения.
Для этого

- В VisualStudio:
В обозревателе решений правой кнопкой на проект -> Cвойста -> Каталоги VC++ -> Включаемые каталоги -> Изменить
Дальше в открывшемся окне в верхнем ряду иконок самая левая (создать строку) и вписать туда каталоги, в которых лежат папки amgcl и boost

Например:
C:\lib\amgcl-1.4.2\
C:\lib\boost_1_78_0\

Свойства ставятся раздельно для Debug и Release конфигураций. Поэтому при переключении убедитесь, что всё поставили.

- Если через Makefile
просто прописать -I{directory} в опции компиллятора

- CMake
include_directories({directory})

5)
Пример использования смотри в файле matsolver.cpp.
Там есть класс SparseMatrix для сборки разряженной матрицы с методом solve().
Тест смотри в функции main(): решение матрицы одномерной диффузии.
Этот файл должен компилироваться и запускаться при выполнении пункта 4.
